  3. If verification fails, recheck USN case (uppercase) and Access Key spelling. Contact mentor if needed.

File Upload Guidelines

  • Allowed: PDF, JPG, PNG. (No ZIPs or links allowed.)
  • Keep under 2 MB per file.
  • Use clear scans/exports; avoid blurry photos.
  • Use descriptive filenames: USN_Category_Title_Sem.pdf
  • One file per activity entry. To add multiple pages, combine into a single PDF.
How to convert Word/Images to PDF
  1. Open in Word/Docs ‚Üí Save as PDF.
  2. On mobile, use a scanner app ‚Üí Export as PDF.
  3. Merge multiple pages using your PDF viewer‚Äôs Combine feature.
